
# 🎓 Study Genius

An AI-powered learning platform that transforms PDFs into summaries, quizzes, flashcards, mind maps, podcasts, and structured study material to make learning faster, smarter, and more interactive.

---

## 📖 About

Study Genius is an intelligent educational platform that helps students convert lengthy academic content into personalized learning resources. Instead of spending hours creating notes, flashcards, questions, and revision material manually, students can upload study documents and instantly receive AI-generated learning assets.

The platform combines document intelligence, generative AI, assessment systems, and personalized study tools into a single learning ecosystem.

## 🚀 Core Workflow

### Step 1: Upload Study Material
Students upload one or more PDFs.

### Step 2: AI Content Processing
The system extracts and analyzes content.

### Step 3: Topic Detection
Important topics and concepts are identified automatically.

### Step 4: Resource Generation

The AI generates:

- Summaries
- MCQs
- Flashcards
- Mind Maps
- Podcasts
- Structured Tables

### Step 5: Interactive Learning

Students can:

- Read summaries
- Review flashcards
- Explore mind maps
- Listen to podcasts
- Attempt quizzes

### Step 6: Performance Analysis
Learning progress is evaluated and stored.
